## 🔴 必须配置（2 项）

### 1️⃣ 配置知识库空间 ID

**为什么需要：** 让机器人知道搜索哪个知识库

**步骤：**

1. 打开飞书，进入你的知识库
2. 从浏览器 URL 复制空间 ID
   - URL 格式：`https://xxx.feishu.cn/wiki/wik/space/SPACE_ID`
   - 复制 `SPACE_ID` 部分（类似 `AbCdEfGhIjKlMn`）

3. 编辑配置文件：
   ```bash
   vi /home/admin/.openclaw/workspace/feishu-helpdesk/config/knowledge-base.json
   ```

4. 填入空间 ID：
   ```json
   {
     "spaces": [
       {
         "id": "这里填你的空间 ID",
         "name": "IT 支持知识库",
         "enabled": true
       }
     ]
   }
   ```

5. **重要**：在飞书知识库中授权机器人
   - 进入知识库 → 设置 → 成员
   - 添加你的机器人为成员（只读权限即可）

---

### 2️⃣ 配置定时任务

**为什么需要：** 让机器人每天自动总结工单

**步骤（任选其一）：**

**方法 A：运行脚本（推荐）**
```bash
bash /home/admin/.openclaw/workspace/feishu-helpdesk/setup-cron.sh
```

**方法 B：手动添加**
```bash
openclaw cron add \
  --name "飞书工单每日总结" \
  --schedule-cron "0 18 * * *" \
  --schedule-tz "Asia/Shanghai" \
  --payload-kind "systemEvent" \
  --payload-text "【定时任务】执行飞书工单每日总结" \
  --session-target "main"
```

**验证：**
```bash
# 查看任务列表
openclaw cron list

# 测试执行（替换 <JOB_ID>）
openclaw cron run --job-id <JOB_ID>
```
